INGAME CONTROLS
Left - Move left
Right - Move right
Up - Move up
Down - Move down
Z or space - Fire
Esc - Quit

VIDEO CONTROLS
X  - Switch between video modes:
     old movie / 3D mode / normal 
F4 - Switch windowed/fullscreen
